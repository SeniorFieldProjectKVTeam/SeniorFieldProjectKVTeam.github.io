Objectives

Our main objective for this project was to create a tool for KnowledgeVision that allows their customers to create customized templates without having to develop front-end development skills and decrease the latency time of engineers for creating the templates for the customers because it gives them a base to work from so some of the repetitive code is unnecessary. This gives the ability to customize templates to customers who want more control over the design of their presentation layout.
